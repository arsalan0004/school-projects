Photo Edit(R) Version 1.0 30/11/2019

Description:
_ _ _ _ _ _ _ _

This is a photo editing software that allows users to load an image
and cumulative apply several different image filters to that image.
The edited image can then be saved onto the user's harddrive.

Instructions:
_ _ _ _ _ _ _ _

This program requires that the user have Python installed. To ensure optimal functionality,
Python versions 3.8.0 and above are recommended. Also Ensure that the following 
modules/modules are stored in the same folder: Cimpl.py, L4_16_image_filters.py,
and L4_16_interactive_ui.py.

General Usage Notes
_ _ _ _ _ _ _ _ _ _ _

Open in the Python shell. Once the program is loaded, the following menu will be shown:

L)oad image S)ave-as
2)-tone 3)-tone X)treme contrast T)int sepia P)osterize
E)dge detect I)mproved edge detect V)ertical flip H)orizontal flip
Q)uit

:
The user will be prompted to input a command. The acceptable commands are the characters
found to the left of the parentheses. The user must first load an image using the "L" input.

e.g

L)oad image S)ave-as
2)-tone 3)-tone X)treme contrast T)int sepia P)osterize
E)dge detect I)mproved edge detect V)ertical flip H)orizontal flip
Q)uit

: L # A file explorer will appear allowing the user to select the image they want to edit
    # If a pop up doesn't appear, minimize all open programs as it might be open in the background.
    # The user can now apply filters to the loaded image.



L)oad image S)ave-as
2)-tone 3)-tone X)treme contrast T)int sepia P)osterize
E)dge detect I)mproved edge detect V)ertical flip H)orizontal flip
Q)uit

: 3 # This will apply the 3- tone filter to the image that was recently loaded.

L)oad image S)ave-as
2)-tone 3)-tone X)treme contrast T)int sepia P)osterize
E)dge detect I)mproved edge detect V)ertical flip H)orizontal flip
Q)uit

: T #A seplia filter will be applied to the already filtered image

L)oad image S)ave-as
2)-tone 3)-tone X)treme contrast T)int sepia P)osterize
E)dge detect I)mproved edge detect V)ertical flip H)orizontal flip
Q)uit

: S #This command will allow the user to save the final image
    # Another file explorer will appear, this will allow the user to select where to save the image and name it.
